# gh-dist

Public distribution channel for **`jira-helper`** — a personal admin CLI for Atlassian Cloud (Jira).

This repository exists **only to serve built artifacts over anonymous HTTPS**, so the
tool can be installed and updated on networks where authenticated GitHub access is
blocked. Its contents are published from a separate source repository — **please don't
edit anything here by hand.**

> ⚙️ The distribution mechanism (artifact index vs. source mirror) is still being set
> up. Install & update instructions will appear here once it is finalized.
